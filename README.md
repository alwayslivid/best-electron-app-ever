# Best Electron App Ever

Silly Electron app made at my own leisure.

## Getting started

* Install Electron.
```
npm install --save-dev electron
```
* Clone the repository.
* Run the command `electron .` inside this directory.

## Made With

* [Electron 4.10](http://electronjs.org)
